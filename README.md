# BOSEDU-App

### Чтобы запустить локальный сервер используйте команды ниже

```
git clone git@github.com:verdx06/BOSEDU-App.git
```
```
cd BOSEDU-App/api
```
```
python3 -m venv venv
```

```
source venv/bin/activate && pip install -r requirements.txt
```

```
python3 -m uvicorn main:app --reload
```
